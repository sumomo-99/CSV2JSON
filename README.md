# CSV2JSON
CSV形式のデータをJSONに変換します。
![demo screenshot 01](docs/images/demo-screenshot-01.png)

## デモURL
[https://csv2json-sumomo-99.vercel.app/](https://csv2json-sumomo-99.vercel.app/)

## 開発サーバの起動
```bash
npm install
npm run dev
```
ブラウザで[http://localhost:3000](http://localhost:3000)にアクセスします。

## ビルド
```bash
npm run build
```
